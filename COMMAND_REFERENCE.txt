================================================================================
🎯 IMPORTANT NOTES
================================================================================

1. ALWAYS run with --dry-run first before production sends
   Example: --mode prod --dry-run

2. Environment variables (.env) must have:
   - EMAIL_ADDRESS=your_gmail@gmail.com
   - EMAIL_PASSWORD=your_app_password

3. The batch send log tracks all sends automatically
   Location: logs/havells_batch_send_log.csv

4. Production mode automatically skips already-sent emails
   This enables safe resumable campaigns

5. Reports are generated in: reports/ directory
   - campaign_report.html (visual dashboard)
   - campaign_report.txt (detailed text report)

6. Test mode uses fixed test emails (6 by default, can override with --test-emails)

7. Batch size: Default is 50, adjust with --batch-size N

8. Delay between batches: Default is 1 second, adjust with --delay-seconds N
