📌 SMART FEATURES IN ACTION
═════════════════════════════════════════════════════════════════════════════

1️⃣  AUTO-COLUMN DETECTION
   Your CSV has: Name, Customer Email, Phone Number, sent_date
   System detects:
      Name     ← name column ✓
      Email    ← Customer Email column ✓
      Phone    ← Phone Number column ✓
      Status   ← sent_date column ✓
   You don't have to map anything manually!

2️⃣  BATCH PROCESSING
   Dataset: 1,245 emails
   System splits into: 25 batches of 50 emails
   Each batch takes ~60 seconds
   Total time: ~25 minutes
   You can cancel anytime (Ctrl+C)
   Next run continues automatically

3️⃣  RATE LIMITING
   0.5 seconds between each email (prevents SMTP blocking)
   5 seconds between batches (recovery time)
   Built-in delays prevent Google blocking you

4️⃣  RESUME FROM ANYWHERE
   Campaign interrupted at Batch 7/25?
   No problem!
   Next run:
      - Detects _WORKING.csv
      - Sees 350 already sent (Status = "Sent")
      - Starts from Batch 8
      - Continues automatically
